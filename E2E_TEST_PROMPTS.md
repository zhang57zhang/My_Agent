# MyAgent 端到端测试提示词

> 覆盖全部 10 个模块、35 个源文件的端到端测试提示词集
> 每条提示词对应一个具体的功能场景，可直接粘贴到 MyAgent 对话中执行

---

## 使用说明

1. 启动 MyAgent：`cd D:\My_Agent\myagent && uv run python -m myagent --provider zhipu --api-key YOUR_KEY`
2. 按顺序执行下方各测试组
3. 对照每个测试的「预期结果」和「验证标准」判断通过/失败
4. 标记失败项，附带错误信息，便于定位问题

```
Python 的 asyncio 和 threading 有什么区别？分别适合什么场景？
```

**预期结果**：
- Agent 识别为 `RESEARCH` 意图

**验证标准**：
